Place Lab was built using open-source and other redistributable components. The HTTP proxy was written by Johannes Plachy and JDBM was developed by Alex Boisvert. The Bluetooth Spotter uses BlueCove, an open-source implementation of the JSR-82. Communication with serial GPS units is handled through rxtx. Hsqldb and MySQL Connector/J are included. The Place Lab distribution also includes SWT and the J9 runtime, both from IBM. Here are the license agreements for these components.
